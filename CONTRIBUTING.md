# Contributing to Trace Guard

We welcome contributions! Whether it's a bug fix, feature addition, or documentation improvement, your help matters.

## How to Contribute

1. **Fork the repo** and create your branch from `main`.
2. **Install dependencies**: `npm install`
3. **Make your changes**. If adding a new detection signal, please follow the kinematic research guidelines in the README.
4. **Test your code**: `npm test`
5. **Lint/Build**: `npm run build`
6. **Submit a Pull Request**.

## Research Principles

Trace Guard is based on kinematic research. When contributing new features:
- Prioritized behavioral signals over static fingerprints.
- Avoid invasive detection methods that impact human privacy.
- Focus on zero-setup UX.

Thank you for making the web safer!
